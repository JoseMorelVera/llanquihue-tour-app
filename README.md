📘 Descripción general del sistema: Este proyecto es para la evaluacion formativa de la semana 6 del curso desarrollo aplicado a objetos, se basa en la continuacion del proyecto LlanquihueTourApp, esta vez agregando las clases de Servicio turistico como clase padre y las calses Ruta gastronominca, Paseo lacustre y Excursion cultural como clases hijas, con la clase gestor datos tomada para crear los obejtso solicitados y la clase main llamando a gestor datos para imprimir en pantalla dos objetos por cada subclase.

🧱 Estructura general del proyecto 📁 src/ ├── UI/ # Clase principal con el método main ├── model/ # Clases de dominio (Cliente, Direccion, Empleado, Guia, Persona, Planes, ServicioTuristico, RutaGastronomica, PaseoLacustre y ExcursionCultural) ├── data/ # Clases para leer datos (GestorDatos)
