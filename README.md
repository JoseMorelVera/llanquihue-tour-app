📘 Descripción general del sistema: Este proyecto es para la evaluacion formativa de la semana 6 del curso desarrollo aplicado a objetos, se basa en la implementacion de lecturas de textos .txt para le empresa Llanquihue tour, generando lista dinamicas para poder mantener el orden dentro de la empresa y no usar metodo escritos a mano que general errores, tambien implementa un metodo para regenerar un menu que se puede interactuar, para poder ver todos los datos de las listas y filtros basicos.

🧱 Estructura general del proyecto 📁 src/ ├── UI/ # Clase principal con el método main ├── model/ # Clases de dominio (Cliente, Direccion, Empleado, Guia, Persona, Planes) ├── data/ # Clases para leer datos (GestorDatos)
